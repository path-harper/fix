Fix Commit Messages

This GitHub App automatically cleans up commit messages by removing "add " prefixes from pushes to installed repositories.

Install the App

Install Fix App on your repository or organization.

How It Works

  • Listens for push events.
  • Fixes commit messages (removes "add ").
  • Force-pushes changes.
  • Skips bot pushes to avoid loops.

Warning

Rewrites Git history. Backup repos. Coordinate with team.
